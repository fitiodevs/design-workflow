# Next.js + Tailwind port smoke — v1.2.0 (T-19)

> First Next.js dogfood of the new adapter pipeline.

## Setup

```bash
npx create-next-app@latest --ts --tailwind --app /tmp/nextjs-smoke
cd /tmp/nextjs-smoke
echo "stack: nextjs-tailwind" > .design-workflow.yaml
```

## Harness

```bash
# 1. Resolve stack (expect: nextjs-tailwind)
STACK=$(python3 ~/design-workflow/scripts/resolve_stack.py)
[ "$STACK" = "nextjs-tailwind" ]

# 2. Emit Plan via /theme-port on the same Figma frame used in T-18
#    Plan saved at /tmp/coupons-plan.json (kind: widget-tree)

# 3. Adapter renders to TSX + globals.css
python3 ~/design-workflow/adapters/nextjs-tailwind/adapter.py /tmp/coupons-plan.json
```
